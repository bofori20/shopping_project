This project uses a Keras neural network trained on e-commerce behavioral data to predict whether a customer will make a purchase. The model was trained on the Online Shoppers Intention Dataset.

It is deployed in a Django web app with a prediction form for user interaction.

  • Administrative: Number of administrative pages visited
  • Administrative_Duration: Time spent on admin pages
  • Informational: Number of informational pages visited
  • Informational_Duration: Time spent on informational pages
  • ProductRelated: Product-related pages viewed
  • ProductRelated_Duration: Time spent on product pages
  • BounceRates: Bounce rate of session
  • ExitRates: Exit rate of session
  • PageValues: Page value estimation
  • SpecialDay: Closeness to a special day (e.g., holiday)
  • OperatingSystems: OS type (numeric)
  • Browser: Browser type (numeric)
  • Region: User region (numeric)
  • TrafficType: Source of traffic (numeric)
  • Weekend: 1 if weekend, else 0

Select values in predict form corresponding to the features. The model will return a prediction indicating whether the user is likely to complete a purchase or not.
